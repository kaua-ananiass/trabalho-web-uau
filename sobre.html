Sobre o projeto

Este projeto foi feito para simular uma loja simples. O usuario escolhe um produto, informa o CEP da entrega, faz a compra e recebe um codigo de rastreio.

Funcoes do site

  • Escolher produto da loja
  • Consultar o endereco pelo CEP na hora da compra
  • Gerar codigo de rastreio
  • Salvar a compra na tabela
  • Rastrear o pedido em uma pagina ficticia
  • Ver preco em outras moedas

APIs usadas

API Uso no projeto
ViaCEP Buscar endereco pelo CEP digitado na compra.
AwesomeAPI Converter o preco dos produtos para outras moedas.
